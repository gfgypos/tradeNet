  1. Summary (System Overview)

  1. Context

The system context diagram provides a general overview for how the system interacts with external entities. For Trade Net, interactions are simple request/response actions between a few basic entities. The circle represents the Trade Net system in whole. Each square in the system context diagram represents an external entity. The arrows between the system and external entities represent all interactions between them.
